## Featured Project

### [AI Workflow Pilot Review](https://github.com/nmatsui7/ai-workflow-pilot-review)

A testable Agent Skill for evaluating proposed AI-assisted workplace
workflows for:

- workflow evidence and business relevance
- appropriate tool selection
- pilot readiness
- human-review requirements
- failure cases and exceptions
- employee workload, trust, and recognition
- governance and accountability
- organizational knowledge accumulation

The project includes a `SKILL.md`, supporting reference files, realistic
business examples, and evaluation cases with expected outcomes.
